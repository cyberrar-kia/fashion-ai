# STYLAURA — AI Fashion Styling Web App

Interactive AI-powered fashion styling app. Users answer 6 questions and receive a fully personalised style profile powered by Claude AI.

## Features
- 6-step map-based question flow
- Real AI recommendations via Claude API
- Style archetype, 3 outfit ideas, capsule wardrobe, 5 expert tips
- Fully responsive (mobile + desktop)
- Dark luxury aesthetic

## Deploy
Push to GitHub → Import on Vercel → Deploy (zero config)
